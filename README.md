# Native American Church of Fletcher, Oklahoma

This project is a simple, grounded website for the Native American Church of Fletcher, Oklahoma.
It is designed to reflect the space as it actually exists — community-centered, relationship-driven, and accessible.

---

## 🧭 Purpose

The site is meant to:

* Share what this space is and how it works
* Help people understand what to expect
* Provide a simple way to connect
* Represent the community in a clear and grounded way
